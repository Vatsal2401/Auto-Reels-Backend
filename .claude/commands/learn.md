# Learn

Capture learnings from the current session and append to learning.md.

## Workflow

1. Invoke `capture-learnings` skill
2. Review recent work (spec, plan, git diff)
3. Extract reusable insights (patterns, gotchas, performance, debugging)
4. Append structured entries to docs in docs/learning/{spec}-{date}-learning.md
5. Extract the insight from this learning and update the docs/learning.md.
6. Report what was captured

## Usage

```
/learn                        # Capture learnings from current session
```
